Allows you to rate bbPress forum topics and replies. Adds bbPress user rating.

== Description ==

This plugin adds compatibility between Rating Manager and bbPress plugins. Now you can easily add rating form to any bbPress topic and reply, and it allows your website visitors to rate every reply. All rating results will be available on statistics page for more convenient way to analyze the rating data for each topic. It also adds bbPress user rating (average and other data).

Rating Manager is the most advanced WordPress content and comment rating plugin. It utilizes SVG image technology and it allows you to modify rating form icons easily without any graphical editor. Basically Rating Manager plugin offers unlimited rating options. Rich snippets for enhanced SEO features, comment rating, comprehensive statistics, feedback form and more functions and features available.

= Features =

* Enables ratings for bbPress topics.
* Enables ratings for bbPress replies.
* Enables rating statistics for rated topics and replies.
* Adds user rating (average and other data).

= Requirements =

* Rating Manager plugin (version 1.0.2 or higher).
* bbPress plugin (version 2.5.8 or higher).

== Installation ==

1. Install and activate Rating Manager and bbPress plugins (you can skip this step if you have Rating Manager and bbPress plugins already installed).
2. Upload `ratings-for-bbpress` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does it work stand-alone? =
No. This plugin requires Rating Manager plugin and bbPress plugin to be installed.

= Can I use it with other WordPress gallery plugins? =
No. This plugin is tailored for bbPress plugin. Rating Manager is capable to deal with all posts, pages, custom posts generated by other plugins and media files, but in some cases it need some minor modifications. This plugin adds necessary modifications for full compatibility between Rating Manager and bbPress plugins.

= Can I change appearance of rating form? =
Yes. Rating Manager allows you to change rating form icons, colors and size of icons right from the settings page. There is no need to use any software for image editing because it uses SVG vector images.
